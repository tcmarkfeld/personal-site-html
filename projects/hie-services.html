HIE Services

Home > Portfolio > HIE Services .NET Microservices

I am currently working for HCA Healthcare on their HIE Services team. HIE Services is essentially the middle man between multiple message senders and receivers. We do the message transformation and API calls to get additional patient information needed for each message.

The current architecture consists of over 200 .NET microservices, which we maintain and update as needed based on business requirements.
We are working on migrating this platform into Apache Nifi, a system built for processing and distributing data. During this migration, we are determining problems with the current architecture and working to optimize it as we move to the new platform.
