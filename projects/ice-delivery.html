Corolla Ice Delivery

Home > Portfolio > Corolla Ice Delivery

This company stored all of their deliveries in a singular Google Sheet. Every Sunday night before the new week started, the driver would go through the sheet and sort all the delveries for the upcoming week so they knew which route to take. This took a while to do, and not all deliveries start on the same day, so it became a lot to keep track of.

I built this app for Corolla Ice Delivery to optimize this process. Instead of using Google Sheets, they now just use the app. The home page has all of the deliveries for that day sorted by neighborhood and address, giving the driver the most optimized route each day. It also has a count at the top so they know how much they need to load in the mornings.
They can now add, edit, and delete deliveries easily. A simple search brings up the delivery they want to edit.

Another issue they had was when it came time to split tips, they had to add up all the tips by hand from Square. This app includes a field for each delivery where they can input the tip amount, and then run a report each month so they know the exact amount that they were tipped in seconds.
